DNSServer

public class DNSServer: DNSResolverDelegate, IPStackProtocol

A DNS server designed as an IPStackProtocol implemention which works with TUN interface.

This class is thread-safe.

    Current DNS server.

    Warning

    There is at most one DNS server running at the same time. If a DNS server is registered to TUNInterface then it must also be set here.
